# Project Documentation EGR115


#This is a simple Rocket Trajectory Calculator created for EGR115

#It prompts you for dry weight, length of rocket, outer diameter of rocket, mass of fuel, drag coefficient, launch angle, burn time, and thrust
#Everything else is calculated from those inputs

#The outputs are: Launch weight, Launch mass, Net Acceleration during burn, Burnout Velocity, Burnout Height 
#Estimated Apogee, Total Flight Time, Estimated Range, and multiple points of the x and y position

#A graph of the projected trajectory is also created
